# Sneha Patel - Portfolio

A premium, AI-driven personal portfolio website built with Next.js, Tailwind CSS, and Framer Motion.
Designed to showcase projects, skills, and experience with a modern, corporate-clean aesthetic.

## Features

*   **Responsive Design**: Mobile-first layout for all devices.
*   **Dynamic Case Studies**: Detailed project pages with architecture diagrams and metrics.
*   **Modern UI**: Glassmorphism effects, smooth animations (Framer Motion), and clean typography.
*   **SEO Optimized**: Sitemap, robots.txt, and metadata included.
*   **Performance**: Optimized with Next.js App Router and Server Components.

## Tech Stack

*   **Framework**: Next.js 15 (App Router)
*   **Styling**: Tailwind CSS
*   **Language**: TypeScript
*   **Animations**: Framer Motion
*   **Icons**: Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run development server:
    ```bash
    npm run dev
    ```
3.  Open [http://localhost:3000](http://localhost:3000).

## Customization

*   **Data**: Edit `lib/data.ts` to update profile, projects, and skills.
*   **Theme**: Edit `tailwind.config.ts` and `app/globals.css` to change colors/fonts.
